### Phase 1：MVP（最简可用版本）

- 功能：销售录入、采购录入、费用录入、简单的利润表
- 数据存储：Excel 或 SQLite
- 目标：能替代手工记账，跑通“收入-成本-费用=利润”

### Phase 2：完善往来管理

- 功能：客户/供应商管理、应收账款/应付账款跟踪、收付款核销
- 新增：往来账龄表（哪些款已超期）

### Phase 3：增加库存管理

- 功能：商品入库/出库、库存预警
- 注意：库存会与采购、销售联动

### Phase 4：多店/多用户 + 报表优化

- 功能：权限控制、多维度报表（按日/月/年、按商品分类）
- 技术：添加用户认证、JWT、部署到云服务器
